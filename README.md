## Getting Started

First, run the development server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- Support webpage screenshots
- Support cloudflare [nopecha](https://nopecha.com/demo)
